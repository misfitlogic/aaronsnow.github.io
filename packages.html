Secure Packages Reuse Project ("SecPrep")

Created to help innovators and implementers in the U.S. government speed up the security clearance process by quickly identifying software packages that have been previously approved. It is a simple list of software and web development tools that have cleared security hurdles and have been deployed in production environments that any knowledgeable person can contribute to.

If you know of a deployed system, please contribute to this list. For now, this means editing the HTML and submitting a pull request. Thank you!
